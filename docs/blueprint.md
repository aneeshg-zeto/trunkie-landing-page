# **App Name**: Trunkie

## Core Features:

- Institutional Management Dashboard: A centralized UI for principals to gain a high-level view of attendance, fee status, and school-wide performance metrics.
- GyanI Exam Engine: A predictive AI tool that automatically generates curriculum-aligned examination papers based on specified topics and difficulty settings.
- Integrated Fee Collection: Digital platform for parents to view due dates and complete online fee payments with automated receipt generation and school-side reconciliation.
- Parent-School Messaging Bridge: A secure communication channel for real-time updates on student progress and school events, bypassing social media clutter.
- Algorithmic Timetable Builder: An automated tool that optimizes class schedules and faculty assignments based on availability and curriculum requirements.
- Student Success LMS: An all-in-one learning management system for distributing resources, assignments, and digital gradebooks for teachers and students.
- GyanI Analytics Layer: A predictive analytics tool that identifies early patterns in attendance and academic decline to support proactive administrative intervention.

## Style Guidelines:

- Primary palette features Navy (#0F1F3D) for authoritative headings and Primary Yellow (#F5C842) for highlights and call-to-action buttons.
- Background scheme utilizes a light, warm approach with Cream (#FAFAF5) and alternating Mist (#F0F2F7) sections for visual depth.
- Headline font: 'Nunito' (Weight 700-900) for high visual impact; Body and UI text: 'Nunito Sans' (Weight 400-600) for professional readability. Note: currently only Google Fonts are supported.
- Modern Lucide icons in line-art style with minimal strokes, used to indicate institutional roles and academic subjects.
- Clean, mobile-first design with high-contrast typographic hierarchy and soft shadow cards (box-shadow: 0 32px 80px rgba(15,31,61,0.12)).
- Subtle, agency-grade smooth scrolling using Lenis and staggered entrance animations powered by Framer Motion (ease-out cubic).
